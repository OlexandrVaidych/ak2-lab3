# AK-2_Lab3

Options for my program:  
-

1. **-l / --list** with required arguments
2. **-v / --variable** with required arguments
3. **-g / --greet**
4. **-h / --help**

Example of usage:
-

./AK2_Lab3 --l 1,2,3 -h --g -v123 -l --h  
  list = 1,2,3  
  Help. Allowed commands:  
  -l / --list  
  -v / --variable  
  -g / --greet  
  -h / --help  
  Hello user!  
  variable = 123
